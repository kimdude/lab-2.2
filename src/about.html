Om webbplatsen

Den här webbplatsen är ett digitalt CV och visar upp min arbetslivserfarenhet. Syftet med webbplatsen är att konsumera ett API som skapades under laboration 1. API:et läses in med Fetch API och använder metoderna GET, POST, PUT och DELETE. Webbplatsen är utvecklad med en automatiserad arbetsprocess i Parcel och SCSS har använts för frontend webbutvecklingen.

API:et bygger på en Postagre databas som publicerats via Render. Samtidigt som det tog ett tag att förstå allt under utvecklingen, var det också förvånansvärt enkelt att skapa. Det krävdes mindre kod än förväntat och, åtminstone REST-webbtjänst, känns ganska rimligt på sitt sätt.

Även om vissa delar varit kluriga har det varit väldigt intressant och lärorikt. Jag kommer att vilja prova att skapa fler API:er och webbplatser som bygger på dem framöver, för att få bättre förståelse för alla delar.
